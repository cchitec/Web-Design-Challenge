Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes
as you get closer to the equator. To accomplish this analysis,
we first pulled data from OpenWeatherMap API to assemble
a dataset on over 500 cities .

After assembling the dataset, we used the Matplotlib to plot
various aspects of the weather vs. latitude. Factors looked at include:
temperature, cloudiness, wind speed, and humidity.
